## 1. 使用 Mermaid 畫圖

Mermaid 支持多種類型的圖表。以下是幾個例子：

### 流程圖
```mermaid
graph TD
    A[開始] --> B{是否繼續?}
    B -->|是| C[進行下一步]
    B -->|否| D[結束]
```

### 思維導圖
```mermaid
mindmap
  root((學習計劃))
    編程語言
      Python
        學習基礎語法
        數據分析
      JavaScript
        前端開發
        後端開發
    數學基礎
      線性代數
      微積分
    工具使用
      Git
        GitHub
        GitLab
      Markdown
        基本語法
        Mermaid 圖表
```
